# Vue 3 + TypeScript + Vite

# Overview
Banx Chat is a modern, responsive chat interface that leverages the power of OpenRouter's AI models to provide intelligent conversations. Built with Vue 3, TypeScript, and Tailwind CSS, it offers a seamless chat experience with support for text-based interactions.

# Key Features
AI-Powered Conversations: Engage with advanced language models through OpenRouter's API
Clean, Modern UI: Intuitive interface with smooth animations and responsive design
Context-Aware: Maintains conversation history for coherent, contextual responses
File Attachments: Supports document and image uploads for enhanced interactions
Customizable Prompts: Predefined templates for common use cases (Draft Prompt, Debug Code, Draft Email)
Dark Theme: Eye-friendly dark mode for extended usage

# Technical Stack
Frontend: Vue 3 with Composition API
Styling: Tailwind CSS with custom theming
State Management: Vuex
Build Tool: Vite
AI Integration: OpenRouter API
